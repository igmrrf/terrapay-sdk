# 🥳 Contributing to terrapay-js

First off, you're awesome! Thank you for even thinking about hanging out and building `terrapay-js` with us. We want to keep this project fun, friendly, and easy to jump into.

## 🌈 How to get involved

There are tons of ways to help out, and they're all super appreciated!

### 🐞 Spot a bug?
If something is broken, let us know! Just open an issue and tell us:
- What happened?
- What did you expect to happen?
- Any spicy error logs or screenshots you've got.

### ✨ Have a cool idea?
Want to see a new feature? Open an issue and let's chat about it! We love fresh ideas.

### 👩‍💻 Want to write some code?
1. **Fork the repo** and make a new branch.
2. **Do your magic!** (If you add a feature, try to add a test so we don't accidentally break it later).
3. **Run the checks**: `bun test` and `bun run lint`.
4. **Send that Pull Request!** We'll review it and give you some high-fives.

## 🛠 Local Setup

We use [Bun](https://bun.sh/) because it's fast and fun.

```bash
bun install    # Get the toys
bun test       # Make sure everything's happy
bun run build  # See the final product
```

## 💬 Questions?
Just open an issue with the `question` label. No question is too small!

Let's build something cool together! 🚀
