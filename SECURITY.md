# 🔒 Security Policy

We care about security, but we also want the process to be friendly and straightforward!

## 🛡 Supported Versions

We keep the latest version of `terrapay-js` patched and happy.

| Version | Status |
| ------- | ------ |
| 1.x     | ✅ Active |

## 🕵️‍♂️ Found a vulnerability?

If you've spotted a security hole, please **don't** post it in a public issue. We'd love to fix it quietly before it causes any trouble!

Instead, shoot an email to [francis.igbiriki@gmail.com].

- We'll get back to you within 48 hours.
- We'll work on a fix together.
- We'll give you a shout-out for helping keep everyone safe!

Thanks for being a security superhero! 🦸‍♂️
